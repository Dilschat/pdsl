pub struct Value<T> { /* fields omitted */ }

A value on the storage.

This is a generic wrapper around a value in the contract storage. It tries to model the wrapped type as close as possible so that using it feels like using the underlying wrapped type.

While optionally implementing several common core traits, such as PartialEq, Debug, Add, ShlAssign, Deref etc. it does not implement DerefMut or IndexMut for security reasons.

For assigning new values or mutating the value inside of it either use set or mutate_with.

[src]

Creates a new storage value for the given key.

Safety

This is an inherently unsafe operation since it does not check for the storage values's invariances, such as

  • Is the storage region determined by the given key aliasing?
  • Is the storage region correctly formatted to be used as storage value?

Users should not use this routine directly if possible.
